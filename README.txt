INTRODUCTION
------------

Image crop module allow user to choose the best cropping to focus
image styles on better displaying.


INSTALLATION
------------

1. Copy the image_crop directory to your sites/all/modules directory.

2. Enable the module at Administer >> Site building >> Modules.
